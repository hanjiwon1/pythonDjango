Process of displaying a static page.

The diagram below is important to understand because it is at the heart of how the Django framework works. It describes the data flows and the components used to process and respond to an HTTP request. We will now get down to:

  • detail the routing of the URLs to associate the views adapted to the HTTP requests that the site will have to process (including with information encoded in the URLs).
  • define the visualization functions and create the HTML pages which will allow the publication of information intended for the users of the site.
  • create the templates that will allow you to publish the data in the views.
